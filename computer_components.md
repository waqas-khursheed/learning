💻 Computer Components – Full Forms aur Roles
| **Short Form**      | **Full Form**                      | **Kya Kaam Karta Hai (Simple Explanation)**                                                                    |
| ------------------- | ---------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **CPU**             | Central Processing Unit            | Computer ka **main brain** – saare instructions (math, logic, control) execute karta hai.                      |
| **RAM**             | Random Access Memory               | **Temporary fast memory** jahan programs run hote hain jab tak computer on hai.                                |
| **ROM**             | Read Only Memory                   | Permanent memory jahan computer ka **boot program (BIOS/firmware)** stored hota hai.                           |
| **SSD**             | Solid State Drive                  | Fast **permanent storage** device (modern hard drive) – data electric chips pe store hota hai.                 |
| **HDD**             | Hard Disk Drive                    | **Older type of permanent storage**, magnetic disks pe data save karta hai.                                    |
| **GPU**             | Graphics Processing Unit           | **Images/videos render karta hai**, pixels draw karta hai, aur screen pe dikhata hai.                          |
| **ALU**             | Arithmetic Logic Unit              | CPU ke andar ka part jo **math (add, subtract)** aur **logic (compare)** operations karta hai.                 |
| **CU**              | Control Unit                       | CPU ke andar ka part jo **data ka flow control** karta hai (kis instruction ko kab run karna hai).             |
| **BIOS**            | Basic Input Output System          | Computer ka **startup program**, jo hardware check karke OS boot karta hai.                                    |
| **OS**              | Operating System                   | **Main software layer** (Windows, Linux, macOS) jo hardware aur user ke beech coordination karta hai.          |
| **I/O**             | Input / Output                     | **Devices** jaise keyboard (input), mouse (input), monitor (output), printer (output).                         |
| **NIC**             | Network Interface Card             | **Internet / network connect** karne wala hardware (LAN/WiFi card).                                            |
| **USB**             | Universal Serial Bus               | **Port standard** jisse keyboard, mouse, flash drive jaisi devices connect hoti hain.                          |
| **LAN**             | Local Area Network                 | **Local network** connection (ghar/office ke andar computers connect karna).                                   |
| **WAN**             | Wide Area Network                  | **Internet level network**, jo multiple LANs ko connect karta hai (jaise worldwide web).                       |
| **DNS**             | Domain Name System                 | Website name (e.g. google.com) ko **IP address** mai convert karta hai.                                        |
| **IP**              | Internet Protocol                  | **Unique address system** har computer/device ke liye jo internet se connect hota hai.                         |
| **HTML**            | HyperText Markup Language          | Web page banane ka **structure language** (text, links, layout define karta hai).                              |
| **CSS**             | Cascading Style Sheets             | Web page ka **design aur styling** control karta hai (colors, fonts, layout).                                  |
| **JS / JavaScript** | (not an acronym, just a name)      | **Web page logic** aur interactivity handle karta hai.                                                         |
| **URL**             | Uniform Resource Locator           | **Web address** ([https://www.youtube.com](https://www.youtube.com)) jo internet resource ko locate karta hai. |
| **HTTP**            | HyperText Transfer Protocol        | Web pages ke **data transfer** ka standard protocol.                                                           |
| **HTTPS**           | HyperText Transfer Protocol Secure | HTTP ka **secure encrypted** version (SSL/TLS ke sath).                                                        |
| **SSL**             | Secure Sockets Layer               | **Data encryption protocol** internet communication secure karne ke liye.                                      |
| **TLS**             | Transport Layer Security           | SSL ka **modern version**, internet communication protect karta hai.                                           |
| **DNS**             | Domain Name System                 | Domain names ko IP addresses mai convert karta hai.                                                            |
