# Phase 2: Vocabulary Building

**Goal:** Roz-marra aur software/client-work se related words itni acchi tarah yaad karo k sochna na pare.

**Tareeqa:** Roz 10 naye words uthao. Ek notebook ya note-taking app me likho:
`English word — Urdu meaning — 1 apna sentence`

---

## 1. Daily Life Words (General)

| Word | Urdu Meaning |
|------|--------------|
| Currently | Abhi, is waqt |
| Available | Available/mojood |
| Regarding | K bary me |
| Regarding this matter | Is mamle k bary me |
| Approximately | Taqreeban |
| Immediately | Foran |
| Recently | Haal hi me |
| Unfortunately | Bad-qismati se |
| Definitely | Zaroor |
| Actually | Asal me |

## 2. Software / Work Vocabulary (Aapke kaam k liye zaroori)

| Word | Meaning | Example |
|------|---------|---------|
| Deploy | Server par live karna | I will deploy the update tonight. |
| Bug | Code ki ghalti | There is a bug in the login form. |
| Feature | Naya function/hissa | The client requested a new feature. |
| Issue | Masla | We found an issue in the payment page. |
| Fix | Theek karna | I fixed the issue in 10 minutes. |
| Update | Taza tar karna/karna | I will update the file. |
| Requirement | Zaroorat/mang | Please share the requirements. |
| Deadline | Aakhri tareekh | The deadline is next Friday. |
| Progress | Pesh raft | Here is the progress report. |
| Pending | Baqi/atka hua | This task is still pending. |
| Scope | Kaam ki hadd | This is out of scope for now. |
| Estimate | Andaza (time/cost) | Can you give me an estimate? |
| Milestone | Aham marhala | We completed the first milestone. |
| Revision | Tarmeem/change | The client asked for a revision. |
| Approval | Manzoori | Waiting for client approval. |

---

## 4. Numbers, Dates, Time (Business me bohot zaroori)

- Dates: "the 5th of July" ya "July 5th"
- Time: "at 3 PM", "by end of day (EOD)", "by tomorrow morning"
- Money: "$500", "five hundred dollars"
- Duration: "It will take two to three days."

---

## 5. Practice Method

1. Har din 10 naye words select karo (upar wali list se).
2. Har word ka 1 sentence khud banao (apni life/kaam se related).
3. Us sentence ko 3 baar zor se bolo.
4. Agle din pichlay din k words revise karo, phir naye 10 add karo.

**1 mahiny me aap ke paas 300 practical words honge — jo kaafi hain client se baat karne k liye.**
