# AI Se Apne Aap Ko Roz Improve Karna

Prompt engineering seekhne ka asal maqsad ye hai ke AI aapki daily growth (coding skills, problem solving, communication) ka ek permanent tool ban jaye — na ke sirf kabhi kabhi copy-paste ke liye use ho.

## AI Ko In 5 Tareeqon Se Use Karo

### 1. Learning Partner Ki Tarah
Naya concept seekhte waqt AI se sirf definition mat pucho — usay apne alfaaz mein explain karo aur AI se check karwao.

> "Maine Dependency Injection ko is tarah samjha hai: {apni samajh likho}. Kya ye sahi hai? Agar kuch missing hai to batao."

Isse aapko pata chalta hai kahan aapki samajh weak hai.

### 2. Debugging Coach Ki Tarah
Jab error aaye, AI se seedha fix mat mangwao — pehle reasoning poocho taake khud bhi seekho.

> "Ye error aa raha hai: {error}. Mujhe fix seedha mat do — pehle mujhe guide karo ke ye error kyun aata hai aur mai khud kaise debug kar sakta hoon."

### 3. Code Reviewer Ki Tarah
Apna khud ka likha code AI ko review ke liye do — regularly, chota chota code bhi.

> "Ye maine khud likha hai. Isay review karo jaise ek senior developer karta — clean code, best practices, aur possible bugs ke nazariye se."

### 4. Interview/Skill Test Ki Tarah
AI se apna khud test lena — knowledge gaps pakadne ka best tareeqa hai.

> "Tum ek technical interviewer ho. Mujhse {topic} pe 5 sawal pucho, easy se hard tak. Har jawab ke baad feedback do."

### 5. Roadmap Planner Ki Tarah
Jab bhi naya skill seekhna ho, AI se ek realistic roadmap banwao (dekho [04_ready_templates.md](04_ready_templates.md) ka Template #6).

## Roz Ka Practice Routine (Suggestion)

| Din Ka Hissa | Kya Karo | Time |
|---|---|---|
| Kaam shuru karne se pehle | Aaj ka goal AI se discuss karo — "aaj mujhe ye feature banani hai, approach suggest karo" | 5-10 min |
| Jab bug aaye | Chain-of-Thought technique se debug karo (dekho [03_prompt_techniques.md](03_prompt_techniques.md)) | as needed |
| Din ke end mein | Aaj likha koi code AI se review karwao | 10-15 min |
| Hafte mein 1-2 dafa | Koi naya concept AI se seekho jo aapke project mein use ho sakta hai | 30-45 min |

## Self-Improvement Ka Mindset

1. **AI se answer mat lo, samajh lo.** Copy-paste karne se aap short-term fast hote ho lekin long-term skill nahi banti.
2. **Apni weak areas AI ko batao.** "Mujhe database optimization weak lagti hai" bol kar targeted help lo.
3. **AI ko apna mirror bana lo.** Apna code, apni understanding, apna approach — sab AI ko dikha kar feedback lo, jaise ek mentor se lete.
4. **Consistency important hai.** Roz thora AI ke sath practice karna, kabhi kabhi lamba session lene se zyada faida deta hai.
5. **Critical thinking mat chhodo.** AI ka har jawab 100% sahi nahi hota — khud verify karna aur sawal uthana seekho, ye khud ek important skill hai.
