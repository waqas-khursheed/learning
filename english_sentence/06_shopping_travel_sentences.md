# Shopping, Restaurant, Travel Aur Directions

## Shopping

| English | Roman Urdu Meaning |
|---|---|
| How much does this cost? | Ye kitne ka hai? |
| Do you have this in a different size? | Kya ye kisi aur size mein hai? |
| Can I try this on? | Kya mai ye try kar sakta hoon? |
| Is there any discount on this? | Kya is par koi discount hai? |
| I'm just looking around, thank you. | Mai bas dekh raha hoon, shukriya. |
| Do you accept card payments? | Kya aap card se payment lete hain? |
| Can I get a receipt, please? | Kya mujhe receipt mil sakti hai? |
| This is too expensive for me. | Ye mere liye zyada mehnga hai. |
| Do you have this in another color? | Kya ye kisi aur color mein hai? |
| I'll take this one. | Mai ye le loonga. |

## Restaurant

| English | Roman Urdu Meaning |
|---|---|
| Can I see the menu, please? | Kya mujhe menu dikha sakte hain? |
| I'd like to order this, please. | Mai ye order karna chahunga. |
| Could we get a table for two? | Kya humein do logon ke liye table mil sakti hai? |
| Is this dish spicy? | Kya ye dish teekhi hai? |
| Can I get the bill, please? | Kya mujhe bill mil sakta hai? |
| This tastes really good. | Ye bohot lazeez hai. |
| Could you bring some water, please? | Kya thora pani la sakte hain? |
| I'm allergic to nuts. | Mujhe nuts se allergy hai. |
| Can we get this to go? | Kya hum ye packing mein le sakte hain? |
| Everything was delicious, thank you. | Sab kuch lazeez tha, shukriya. |

## Travel / Directions

| English | Roman Urdu Meaning |
|---|---|
| Excuse me, how do I get to the station? | Maaf kijiye, station kaise jaun? |
| Is this the right way to the airport? | Kya ye airport jane ka sahi raasta hai? |
| How far is the nearest bus stop? | Sabse qareebi bus stop kitni door hai? |
| Can you show me on the map? | Kya aap mujhe map par dikha sakte hain? |
| Turn left at the next signal. | Agle signal par bayen mur jayen. |
| Go straight and then turn right. | Seedha jayen phir dayen mur jayen. |
| It's just around the corner. | Ye bas mor ke pass hi hai. |
| How long does it take to get there? | Wahan pohanchne mein kitna waqt lagta hai? |
| I'm lost, can you help me? | Mai raasta bhool gaya hoon, kya madad kar sakte hain? |
| What's the fastest way to reach downtown? | Downtown pohanchne ka sabse tez raasta kya hai? |
