# Roz Poochay Jane Wale Sawal (Common Questions)

## General Questions

| English | Roman Urdu Meaning |
|---|---|
| What are you doing right now? | Aap abhi kya kar rahe hain? |
| Where are you going? | Aap kahan ja rahe hain? |
| What time is it? | Kitne baje hain? |
| What time does it start? | Ye kis waqt shuru hota hai? |
| How far is it from here? | Ye yahan se kitni door hai? |
| How long will it take? | Isay kitna waqt lagega? |
| What's your opinion on this? | Is bare mein aapki kya raay hai? |
| Can you help me with this? | Kya aap is mein meri madad kar sakte hain? |
| Do you have a minute? | Kya aapke paas ek minute hai? |
| Is everything okay? | Sab theek hai? |
| What happened? | Kya hua? |

| How does this work? | Ye kaise kaam karta hai? |
| What's the best way to do this? | Ye karne ka behtareen tareeqa kya hai? |

## Clarification Mangna

| English | Roman Urdu Meaning |
|---|---|
| Could you please repeat that? | Kya aap dobara bata sakte hain? |
| What do you mean by that? | Aap iska kya matlab keh rahe hain? |
| Can you explain it differently? | Kya aap isay kisi aur tarah samjha sakte hain? |
| Sorry, I didn't quite understand. | Maafi chahta hoon, mujhe theek se samajh nahi aaya. |
| Are you saying that...? | Kya aap ye keh rahe hain ke...? |
| Just to confirm, you mean...? | Sirf confirm karne ke liye, aapka matlab hai...? |
| Can you give me an example? | Kya aap mujhe ek example de sakte hain? |
