# Greetings, Introductions Aur Small Talk

## Milne Par (Meeting Someone)

| English | Roman Urdu Meaning |
|---|---|
| Hi, how are you doing? | Hi, aap kaisay hain? |
| I'm doing great, thanks for asking. | Mai bohot achha hoon, pochne ka shukriya. |
| Nice to meet you. | Aap se milkar khushi hui. |
| It's a pleasure to meet you. | Aap se milna baais-e-khushi hai. |
| Long time no see! | Bohot dinon baad mile! |
| How have you been? | Aap kaisay rahe (in dino)? |
| I've been good, and you? | Mai theek raha, aur aap? |
| It's great to see you again. | Aapko dobara dekh kar accha laga. |
| What have you been up to lately? | Aap in dino kya kar rahe hain? |
| We haven't talked in a while. | Humne kaafi arse se baat nahi ki. |

## Apna Taruf Karwana (Introducing Yourself)

| English | Roman Urdu Meaning |
|---|---|
| My name is Waqas. | Mera naam Waqas hai. |
| I'm a web developer. | Mai ek web developer hoon. |
| I work as a Laravel developer. | Mai Laravel developer ke tor par kaam karta hoon. |
| I've been working in this field for three years. | Mai is field mein teen saal se kaam kar raha hoon. |
| I'm currently learning new skills. | Mai abhi naye skills seekh raha hoon. |
| Let me introduce myself. | Mujhe apna taruf karwane dein. |
| This is my colleague, Ahmed. | Ye mere colleague hain, Ahmed. |
| I'd like you to meet my friend. | Mai chahta hoon aap mere dost se milen. |

## Small Talk (Aam Baat Cheet)

| English | Roman Urdu Meaning |
|---|---|
| How's everything going? | Sab kuch kaisa chal raha hai? |
| How was your weekend? | Aapka weekend kaisa raha? |
| Did you have a good day? | Aapka din accha guzra? |
| What do you do for a living? | Aap kya kaam karte hain? |
| Where are you from? | Aap kahan se hain? |
| How long have you lived here? | Aap yahan kitne arse se rehte hain? |
| Do you have any plans for the weekend? | Weekend ke liye koi plan hai? |
| I'm just relaxing at home these days. | Mai aajkal bas ghar pe relax kar raha hoon. |
| That sounds interesting. | Ye kaafi interesting lagta hai. |
| I totally agree with you. | Mai aap se poori tarah mutafiq hoon. |

## Vida Lena (Saying Goodbye)

| English | Roman Urdu Meaning |
|---|---|
| It was nice talking to you. | Aap se baat kar ke accha laga. |
| I have to go now. | Mujhe ab jana hai. |
| Take care of yourself. | Apna khayal rakhna. |
| See you soon. | Jald milte hain. |
| Let's catch up again sometime. | Kabhi phir milte hain. |
| I'll talk to you later. | Mai baad mein baat karta hoon. |
| Have a great day ahead. | Aapka din accha guzre. |
| Stay in touch. | Rabta rakhna. |

## Shukriya Aur Maafi (Thanks and Apologies)

| English | Roman Urdu Meaning |
|---|---|
| Thank you so much for your help. | Aapki madad ke liye bohot shukriya. |
| I really appreciate it. | Mai isay bohot sarahta hoon. |
| No problem at all. | Koi masla nahi. |
| You're welcome. | Koi baat nahi (khushi hui madad kar ke). |
| I'm sorry for the inconvenience. | Museebat ke liye maafi chahta hoon. |
| I apologize for being late. | Der se aane ke liye maafi chahta hoon. |
| It's completely fine. | Ye bilkul theek hai. |
| Don't worry about it. | Iski fikar mat karein. |
